# Import Inbox

Use this folder for raw source material that should eventually become part of Jarvis's local brain.

Connected notes: [[05 Source Inventory]], [[03 Brain Schema]], [[04 Capture Workflow]], [[50 Synthesis]]

Purpose: keep raw provenance traceable while routing durable facts into the curated brain.

Keep imports separated by source so they stay traceable:

- `claude/` for Claude exports
- `chatgpt/` for ChatGPT exports
- `openai/` for OpenAI API or other OpenAI-related exports if needed

Recommended workflow:

1. Drop the raw export here unchanged
2. Add a short provenance note if the export format is not obvious
3. Distill durable facts, preferences, projects, and summaries into [[10 Identity]], [[20 Projects]], [[50 Synthesis]], [[60 Interview Story Bank]], or [[61 Role Targeting Map]]
4. Avoid treating raw transcripts as the final knowledge layer
